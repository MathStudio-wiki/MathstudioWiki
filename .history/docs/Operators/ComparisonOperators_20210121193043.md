## 比较运算符

以下这些比较运算符分别返回1或0来表示该语句是正确的还是错误的。

- 等于        (a == b)
- 不等于    (a != b)
- 大于        (a > b)
- 小于        (a < b)
- 大于等于 (a >= b)
- 小于等于 (a <= b)
