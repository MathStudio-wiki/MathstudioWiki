### 特殊值
特殊值(例如$\pi$)，通过使用 "@" 符号进行表示。

### Pi
> ```math
> @pi
> ```
>
> $\pi$
>
> 3.14159265359

> ```math
> +@pi
> ```
>
> $3.14159265359$

> ```math
> +@pi
> ```
>
> $6.28318530718$
