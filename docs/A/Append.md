### Append(*expression1, expression2*)

将一个表达式附加到另一个表达式的末尾。

### 示例

[在MathStudio中浏览](http://mathstud.io/?input[0]=QXBwZW5kKFsxLDIsM10sNCk%3D&input[1]=QXBwZW5kKGFiLGMp&input[2]=QXBwZW5kKFsxLDIsM10sWzQsNSw2XSk%3D&input[3]=QXBwZW5kKGEsMSk%3D)

> ```math
> Append([1, 2, 3], 4)
> ```
>
> **[1, 2, 3, 4]**

> ```math
> Append(ab, c)
> ```
>
> ***abc***

> ```math
> Append([1, 2, 3], [4, 5, 6])
> ```
>
> **[1, 2, 3, 4, 5, 6]**

> ```math
> Append(a, 1)
> ```
>
> ***a1***
