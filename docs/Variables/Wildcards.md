### 通配符

通配符(~)通常用于比较表达式中。

### 例子

> ```math
> [1, 2, 3, 4, 5] == [1, 20, 30, 4, 5]
> ```
>
> $0$

> ```math
> [1, 2, 3, 4, 5] == [1, ~, ~, 4, 5]
> ```
>
> $1$

> ```math
> x^2 == a^2
> ```
>
> $0$

> ```math
> x^2 == ~^2
> ```
>
> $1$

> ```math
> x^2 + a^2 == x^2 + y^2
> ```
>
> $0$

> ```math
> x^2 + ~^2 == x^2 + y^2
> ```
>
> $1$

> ```math
> cos(x) == cos(y)
> ```
>
> $0$

> ```math
> cos(x) == cos(~)
> ```
>
> $1$
