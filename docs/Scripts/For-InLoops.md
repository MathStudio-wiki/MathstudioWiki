### 遍历列表

> ```math
> numbers = 1:10
> sum = 0
> for number in numbers
>  sum += number
> end
> sum
> ```
>
> $55$

### 遍历范围

> ```math
> sum = 0
> for i in 1..10
>  sum += i
> end
> sum
> ```
>
> $55$

### 使用箭头操作符更改范围的步长

> ```math
> sum = 0
> for i in 1..10 -> 2
>  sum += i
> end
> sum
> ```
>
> $25$

### 使用嵌套循环遍历矩阵

> ```math
> data = @[29, 29]
> for row in 1..29
>  for column in 1..29
>   data(row, column) = gcd(row, column) > 1
>  end
> end
> ImagePlot(data)
> ```

### 使用嵌套循环遍历多个矩阵

> ```math
> matrices = @[3, 4, 5]
> for i in 1..3
>  for j in 1..4
>   for k in 1..5
>    matrices(i, j, k) = i + j + k
>   end
>  end
> end
> matrices
> ```
