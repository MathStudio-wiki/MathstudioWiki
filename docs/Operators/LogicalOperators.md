## 双幂运算符

逻辑运算符非(!)是一元运算符，如果表达式等于0，则返回1，否则返回0。

逻辑操作符与(&& 或写作 &)如果两个表达式都不为零，则返回1，否则返回0。

逻辑操作符或(|| 或写作 |)如果其中一个表达式不为零，则返回1，否则返回0。

## 逻辑NOT

> ```math
> a=0
> !a
> ```
>
> $1$

> ```math
> a=1
> !a
> ```
>
> $0$

## 逻辑AND

> ```math
> 1 && 2
> ```
>
> $1$

> ```math
> 1 && 0
> ```
>
> $0$

## 逻辑OR

> ```math
> 1 || 2
> ```
>
> $1$

> ```math
> 1 || 0
> ```
>
> $1$
