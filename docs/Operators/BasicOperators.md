## 基本运算符  
运算符是检查、更改或组合值的特殊符号。例如，加法运算符(+)将两个表达式相加。

## 加法  


>    ```math
>    1 + 2 + 3
>    ```
>    ${\text{6}}$


>    ```math
>    a + b + b + c + c + c
>    ```
>    $a + 2b + 3c$


## 减法
>    ```math
>    12 - 6
>    ```
>    ${\text{6}}$

>    ```math
>    5x - 2x
>    ```
>    ${\text{3}}x$

## 乘法
>   ```math
>   12 * 5
>   ```
>   $60$

>   ```math
>   x*x + y*y*y
>   ```
>   ${x^2} + {y^3}$

## 除法
>   ```math
>   20 /5 
>   ```
>   $4$

>   ```math
>   x^4 / x
>   ```
>   ${x^3}$

## 取余    
>   ```math
>   20 % 3
>   ```
>   $2$

>   ```math
>   12.5 % 10
>   ```
>    
>   $2.5$
（注：根据mathstudio的设置不同也可能显示为$\frac{5}{2}$）


## 一元负号
>   ```math
>   -3
>   ```
>   $ - 3$

>   ```math
>   -(x+y)
>   ```
>   $ - x - y$

## 一元正号
>   ```math
>   +@pi
>   ```
>   $3.14159265359$

>   ```math
>   +(2@pi)
>   ```
>   $6.28318530718$

>   ```math
>   +sin(1)
>   ```
>   $0.841470984808$

>   ```math
>   +"12"
>   ```
>   $12$

## 阶乘
>   ```math
>   6!
>   ```
>   $720$
