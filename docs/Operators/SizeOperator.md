## 大小运算符

大小运算符(\$)是一元运算符，用于返回表达式的大小。



## 举例

[在Mathstudio上浏览](http://mathstud.io/?input[0]=JGFiYw%3D%3D&input[1]=JCJIZWxsbyBXb3JsZCEi&input[2]=JFsxLDIsMyw0LDVd&input[3]=JFtbMSwyLDNdLFs0LDUsNl1d)


## 变量

> ```math
> $abc
> ```
>
> $3$

> ```math
> $"Hello World!"
> ```
>
> $12$

## 列表

> ```math
> $[1, 2, 3, 4, 5]
> ```
>
> $5$


## 矩阵

> ```math
> $[[1, 2, 3], [4, 5, 6]]
> ```
>
> $2$
