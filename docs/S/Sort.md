## Sort(list)

以升序对列表进行排序。


## 举例  
[在Mathstudio上浏览](http://mathstud.io/?input[0]=U29ydChbMywyLDFdKQ%3D%3D&input[1]=U29ydChbMiw1LDMsNCwxXSk%3D)




>   ```math
>   Sort([3, 2, 1])
>   ```
>   $ [1,2,3] $



>   ```math
>   Sort([2, 5, 3, 4, 1])
>   ```
>   $ [1,2,3,4,5]$
