## Size(list)
返回列表或矩阵的大小。



## 举例  
[在Mathstudio上浏览](http://mathstud.io/?input[0]=c2l6ZShbMSwyLDMsNCw1LDZdKQ%3D%3D&input[1]=c2l6ZShbWzEsMiwzXSxbNCw1LDZdXSk%3D)




>   ```math
>   size([1, 2, 3, 4, 5, 6])
>   ```
>   $ 6$



>   ```math
>   size([[1, 2, 3], [4, 5, 6]])
>   ```
>   $ [2,3] $
