## Sign(z)

返回给定值的符号。


如果z是复数，则符号返回z/abs(z)。


## 举例

[在Mathstudio上浏览](http://mathstud.io/?input[0]=c2lnbigxLjIzNCk%3D&input[1]=c2lnbigtOS44NzY1KQ%3D%3D&input[2]=c2lnbigwKQ%3D%3D&input[3]=c2lnbig3LzIrNy8zQGkp)




>   ```math
>   sign(1.234)
>   ```
>   $ 1 $

>   ```math
>   sign(-9.8765)
>   ```
>   $ -1 $

>   ```math
>   sign(0)
>   ```
>   $ 0 $

>   ```math
>   sign(7/2+7/3@i)
>   ```
>   $ 0.832050294338-0.554700196225i $
